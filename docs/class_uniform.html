Detailed Description

Concrete crossover strategy class. Implements the Crossover strategy.

In this crossover method each gene (city) is selected randomly from one of the corresponding genes of the parent chromosomes (paths). Cross method creates children, performs the crossover on the parents (using helper uniformCross function) and returns the result of this operation.
