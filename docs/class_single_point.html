Detailed Description

Concrete crossover strategy class. Implements the Crossover strategy.

Random crossover point is selected and the tails (cities after that point) of its two parents paths are swapped to get new paths (children). Cross method creates children, pick the crossover point, performs the crossover on the parents (using helper singleCross function) and returns the result of this operation.
